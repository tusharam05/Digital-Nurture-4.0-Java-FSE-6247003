# GitDemo Project

## Step 1: Initialize Git
```bash
git init
```

## Step 2: Check status
```bash
git status
```

## Step 3: Add file
```bash
git add welcome.txt
```

## Step 4: Commit file
```bash
git commit -m "Initial commit with welcome.txt"
```

## Step 5: Push to remote
```bash
git remote add origin <your-remote-repo-url>
git push -u origin master
```
